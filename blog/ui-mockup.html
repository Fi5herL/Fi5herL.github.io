Table 23 - Creepage distances (mm)

The system determined the required value by cross-referencing the working voltage with pollution degree and material group.

Working Voltage (V) Pollution Degree / Material Group
1 2/I 2/II 2/IIIa 2/IIIb 3/I 3/II 3/IIIa 3/IIIb
50 0.2 0.6 0.9 1.2 1.5 1.7 1.9 2.0 2.5
100 0.3 0.8 1.1 1.5 1.8 2.2 2.4 2.5 3.2
150 0.4 1.0 1.4 1.8 2.2 2.8 3.0 3.2 4.0
200 < V ≤ 250 0.6 1.3 1.8 4.0 3.0 3.6 3.8 4.0 5.0

Decision Process

1
Input Parameters: Working Voltage = 240V, Pollution Degree = 2, Material Group = IIIa
↓
2
Table Lookup: For 200 < V ≤ 250, PD2/MGIIIa, the required value is 4.0 mm
↓
3
Comparison: Measured value (3.5 mm) < Required value (4.0 mm)
↓
4
Result: FAIL - The measured creepage distance does not meet the minimum requirement

Recommendation

Increase the creepage distance between Line and Neutral on the PCB to at least 4.0 mm by:

  • Redesigning the PCB layout with increased spacing
  • Adding physical barriers or slots between conductors
  • Using higher CTI material (Material Group I or II)
